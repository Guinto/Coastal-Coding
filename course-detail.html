Coastal Coding Curriculum

Front End Development

The first 2-3 weeks will be dedicated to learning the fundamentals of web development using HTML5 and CSS3. During this time you will develop websites from scratch using the skills you would have learnt.

The next 4 weeks will delve deep into Javascript. This is the only scripting language that is used both in the client and server side. Therefore, it is of utmost importance that you learn it. The most important skills you should walk out of the door with include web design principles, creating and handling forms, creating Javascript cookies, making your pages dynamic, and scripting mobile devices.

Back End Development

The next 5 weeks will be spent learning how to program with PHP, creating dynamic websites. This involves building your own Shopping Cart, Message Boards and creating Cookies and Sessions. Needless to say, you will be fully immersed in MySQL creating your own database and executing queries thereof.

The skills that you should walk out of the door with will include writing your own scripts, Relational Database Normalization, and writing queries to access, add, modify or delete data from your database. You will also learn how to register a user, activate an account, create a shopping cart, product catalogue and record orders.
