Coastal Coding Curriculum

Front End Development

The first 2-3 weeks will be dedicated to learning the fundamentals of web development using HTML5 and CSS3. During this time you will develop websites from scratch using the skills you would have learnt.

The next 4 weeks will delve deep into Javascript. This is the only scripting language that is used both in the client and server side. Therefore, it is of utmost importance that you learn it. The most important skills you should walk out of the door with include web design principles, creating and handling forms, creating Javascript cookies, making your pages dynamic, and scripting mobile devices.

Back End Development

The next 5 weeks will be spent learning MAMP (Mac OS X, the operating system; Apache, the web server; MySQL, the database management system; and PHP). By this time you will be working on an Ecommerce or a content management project.

You will have a functional database that you will be able to update and modify using SQL. The skills that you should walk out of the door with will include writing your own scripts, relational database normalization, and writing queries to access, add, modify or delete data from your database. You will also learn how to register a user, activate an account, create a shopping cart, product catalogue and record orders.

Essential Skills

From the very first week of class you will be introduced to GitHub, repository that provides a Web-based graphical interface and desktop as well as mobile integration. GitHub is a powerful tool that provides access control and several collaboration features such as bug tracking, feature requests, and task management.

This will prepare you for large company projects that you will be tasked with as part of a team of developers. We will also teach you debugging skills and various methods that will train you to think critically and methodically to solve real life programming tasks. Basically your vocabulary is going to be transformed to some geeky lingua franca.

Projects & Portfolio

Your portfolio will consist of multiple fronted based websites for those customers and companies that just want to display information. Your team project will showcase a backend website endowed with everything that you learnt in the frontend design.

This time your website will fully interact with the server and and enable your users to blog in different languages and and order merchandize via your shopping cart.
